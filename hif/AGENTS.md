# hif Agent Notes

`hif skill` is the canonical agent-facing workflow document for the Micelio CLI.

When you change `hif` capabilities, command behavior, or how repositories are versioned with Micelio sessions and landing, update the source skill content at `app/priv/static/SKILL.md`.

The `hif skill` command prints that file directly, so keeping it current keeps CLI-discoverable agent guidance current too.
